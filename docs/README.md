# CineFind online JSON data

Dit mapje wordt gepubliceerd via GitHub Pages.

De app leest `hotspots.json` niet lokaal uit de projectmap, maar haalt deze online op met `fetch` via:

https://quinten-1074726.github.io/CineFind/hotspots.json

Hiermee voldoet de app aan de eis dat de hotspots uit JSON-data van een online bron komen.
